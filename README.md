# PC28 策略回测

过去 1 年 PC28 真实开奖数据上的多策略回测可视化。

## 数据
- 数据源: PC28 30 年完整历史 (BCLC Keno 算法衍生),取过去 1 年子集
- 期数: 146,396 期 (2025-04-29 → 2026-04-28)

## 资金管理 (硬规则)
- 起始 $10,000 → 口袋 $2,000 + 备用金 $8,000
- 爆仓 (口袋归零) → 从备用金拿 1/5 当新口袋
- 总额达到锚点 ×2 → 重新分配 (1/5 + 4/5),锚点更新
- 总额 ≥ $1,000,000 → 停手锁仓
- 限红: 单注 1-12,000

## 添加新策略
1. 编辑 `strategies/run_all.py`,在 `STRATEGIES` 列表加新条目
2. 实现策略函数 (返回 `(side, amount, reason)` 或 `None`)
3. 跑 `python3 strategies/run_all.py` 重新生成 JSON
4. `git add data/ && git commit -m "..." && git push` 自动部署

## 数学事实
PC28 是真随机游戏 (BCLC Keno 衍生),理论 EV ≈ -0.15%。任何策略长期都会输,短期波动取决于运气。
